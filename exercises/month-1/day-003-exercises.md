## 🎧 LISTENING TEXT — Day 3

### Listening 3.1: What is Programming? (Chậm)

```
Programming is the process of writing instructions for a computer. These instructions are called code. A programmer writes code using a programming language, such as JavaScript, Python, or Java.

When you write code, you use variables to store information. A variable is like a box that holds a value. For example, you can create a variable called "name" and put the value "John" inside it.

You also use functions. A function is a group of instructions that does a specific job. You can call a function whenever you need it. This saves time because you do not have to write the same code again and again.

Loops are another important concept. A loop repeats a block of code many times. For example, if you want to print the numbers one to ten, you can use a loop instead of writing ten separate lines of code.

When your code has a mistake, we call it a bug. The process of finding and fixing bugs is called debugging. All programmers spend a lot of time debugging their code.
```

### Listening 3.2: A Simple Program (Bình thường)

```
Let me explain how to write a simple program in JavaScript.

First, I create a variable called "message" and set its value to "Hello, World." Then, I use console dot log to display the message on the screen.

Next, I want to add two numbers. I declare a variable called "a" and set it to five. I declare another variable called "b" and set it to three. Then I create a third variable called "result" and set it to a plus b. When I log the result, the output is eight.

Now, let me show you a function. I define a function called "greet" that takes a name as input. Inside the function, I return the string "Hello" followed by the name. When I call greet with the argument "Alice," the output is "Hello, Alice."

Finally, I use a for loop to print numbers from one to five. The loop starts at one, checks if the number is less than or equal to five, and then adds one each time. The output is: one, two, three, four, five.
```

### Listening 3.3: Dictation

```

Use a loop to repeat an action.
An array can hold many items.
Debug your code to find errors.

The boolean value is true or false.
```
