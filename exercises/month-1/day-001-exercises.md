## 📝 BÀI TẬP 1: Phát Âm /iː/ vs /ɪ/

### Exercise 1.1: Nghe và phân biệt
Nghe đoạn audio sau. Mỗi câu có 1 từ trống — điền từ bạn nghe được:

1. Please take a _____ (seat / sit)
2. I need to _____ the code (read / rid)
3. The bug is in the _____ (heap / hip)
4. Can you _____ the log? (see / sip)
5. This _____ is important (feature / fitter)
6. Press the _____ (key / kit)
7. Add it to the _____ (list / least)
8. The ship is in the _____ (sea / —)

**Đáp án:** 1. seat, 2. read, 3. heap, 4. see, 5. feature, 6. key, 7. list, 8. sea

### Exercise 1.2: Đọc to và ghi âm
Đọc mỗi cặp từ 3 lần, nhấn mạnh sự khác biệt:
1. beat /biːt/ — bit /bɪt/
2. sheep /ʃiːp/ — ship /ʃɪp/
3. feel /fiːl/ — fill /fɪl/
4. leave /liːv/ — live /lɪv/
5. team /tiːm/ — Tim /tɪm/

Ghi âm → nghe lại → bạn có phân biệt rõ không?

### Exercise 1.3: Điền /iː/ hoặc /ɪ/
Xác định từ nào chứa /iː/, từ nào chứa /ɪ/:

| Từ | /iː/ hay /ɪ/? |
|---|---|
| screen | ? |
| fix | ? |
| team | ? |
| git | ? |
| release | ? |
| bit | ? |
| delete | ? |
| system | ? |

**Đáp án:** screen /iː/, fix /ɪ/, team /iː/, git /ɪ/, release /iː/, bit /ɪ/, delete /iː/, system /ɪ/

## 📝 BÀI TẬP 2: Từ Vựng

### Exercise 2.1: Matching — Nối từ với nghĩa

| | Từ | | Nghĩa |
|---|---|---|---|
| 1 | computer | a | tải xuống |
| 2 | keyboard | b | mật khẩu |
| 3 | software | c | máy tính |
| 4 | download | d | bàn phím |
| 5 | password | e | phần mềm |
| 6 | network | f | tệp |

| 8 | upload | h | tải lên |

**Đáp án:** 1-c, 2-d, 3-e, 4-a, 5-b, 6-g, 7-f, 8-h

### Exercise 2.2: Fill in the blanks
Điền từ phù hợp:

1. I need to change my _________ because it is too weak. (password)
2. Please _________ the file to the server. (upload)
3. Open the _________ and type the URL. (browser / chưa học → cho phép "internet")
4. Save the document in a new _________. (folder)
5. Check your _________ for the meeting invitation. (email)

### Exercise 2.3: Dịch sang tiếng Anh
1. Tải phần mềm xuống. → _________
2. Lưu tệp vào thư mục. → _________
3. Kết nối với mạng wifi. → _________
4. Đổi mật khẩu của bạn. → _________
5. Mở trang web. → _________

**Đáp án:** 1. Download the software. 2. Save the file to the folder. 3. Connect to the wifi network. 4. Change your password. 5. Open the website.

## 🎧 LISTENING TEXT — Day 1
*Chuyển đoạn text dưới đây thành audio (dùng TTS) để luyện nghe*

### Listening 1.1: Self Introduction (Tốc độ chậm)

```
Hello. My name is David. I am a software developer. I work with computers every day.

I use a keyboard and a mouse to write code. My computer has a big screen. I save my files in folders on my computer.

Every morning, I check my email. Then I download the latest software updates. I also need a strong password to keep my work safe.

I connect to the internet through a network. I visit many websites to learn new things. Sometimes I upload my work to the company server.

I like my job because I can create useful software for people.
```

### Listening 1.2: IT Basics (Tốc độ bình thường)

```
Welcome to your first day at the company. Let me show you how to set up your computer.

First, turn on the computer and wait for it to start. You will see the login screen. Enter your username and password. Make sure your password is strong — use numbers, letters, and special characters.

Next, connect to the company network. Click on the wifi icon and select our network name. You will need the network password. I will give it to you.

After that, open your email and check for any messages from the team. You should also download and install the software we use for work.

Finally, create a folder on your desktop to save your project files. Keep everything organized. If you have any questions, send me an email.
```

### Listening 1.3: Dictation Exercise (Nghe và viết lại)

```
My computer has a keyboard, a mouse, and a screen.
I save my files in a folder.
I download software from the internet.
Please change your password every month.
Send me an email with the file.
```

## 📝 BÀI TẬP 4: Nói

### Exercise 4.1: Read aloud
Đọc to đoạn sau, chú ý /iː/ và /ɪ/:

> "I need to **see** the **screen**. The **team** will **fix** the **system**. Please **delete** the **bit** that is not needed. The new **feature** is a **big** improvement."

Ghi âm → nghe lại → tự đánh giá.

### Exercise 4.2: Describe your setup
Nhìn quanh bàn làm việc → mô tả bằng tiếng Anh:
- "I have a [computer/laptop]. It has a [size] screen..."
- "I use a [type] keyboard and a [type] mouse..."
- "My files are saved in..."

Ghi âm 1 phút.
