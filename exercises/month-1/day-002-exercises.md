## 📝 BÀI TẬP 1: Phát Âm /e/ vs /æ/

### Exercise 1.1: Phân loại âm
Xếp các từ vào đúng cột:

**Từ:** test, app, web, class, help, batch, send, access, error, handle

| /e/ | /æ/ |
|---|---|
| ? | ? |

**Đáp án:** /e/: test, web, help, send, error | /æ/: app, class, batch, access, handle

### Exercise 1.2: Cặp tối thiểu — Đọc và ghi âm
1. set /set/ → sat /sæt/
2. bed /bed/ → bad /bæd/
3. pen /pen/ → pan /pæn/
4. men /men/ → man /mæn/
5. bet /bet/ → bat /bæt/

Đọc mỗi cặp 5 lần → ghi âm lần cuối.

### Exercise 1.3: Chọn từ đúng
1. The developer (set / sat) up the server. → **set**
2. I need to (send / sand) the email. → **send**
3. This is a (bed / bad) error. → **bad**
4. The (men / man) fixed the bug. → **man**
5. I (bet / bat) this will work. → **bet**

## 🎧 LISTENING TEXT — Day 2

### Listening 2.1: Daily routine of a developer (Chậm)

```
Good morning. I am a web developer and this is my daily routine.

I start my day at nine o'clock. First, I open my laptop and type my password. Then I click on my email to check for new messages.

After that, I open my code editor. I type code and save my work often. When I make a mistake, I press Control Z to undo it. If I need to copy code, I select it and press Control C. Then I paste it with Control V.

Sometimes I need to delete old files to clean up my computer. I search for files I do not need and delete them.

At the end of the day, I save everything and close all my applications. Then I restart my computer to install updates.
```

### Listening 2.2: Setting up a new project (Bình thường)

```
Today I am going to set up a new project. Let me walk you through the steps.

First, I open the terminal and type a command to create a new folder. I call it "my-project." Then I open the folder in VS Code.

Next, I need to install some software packages. I type "npm install" and wait for it to finish. This downloads all the code libraries I need.

After the installation is complete, I create a new file called "index.js." I open the file and start typing my code. I write a simple function that says "Hello, World."

I save the file and run the program by typing "node index.js" in the terminal. The output appears on the screen: "Hello, World."

Great! The project is working. Now I can start building the real features.
```

### Listening 2.3: Dictation (Nghe từng câu, viết lại)

```
Click the button to open the file.
Type your email address in the box.
Save your work before you close the window.
Delete the old folder from your computer.
Install the software update now.
Copy the link and paste it in the browser.
Search for the answer on the website.
Run the program and check the output.
```
