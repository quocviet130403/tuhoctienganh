# Day 21: Ôn Tập Tuần 3 + Mini Presentation

> 🎯 **Mục tiêu:** Ôn tập trọng âm, ngữ điệu, connected speech. Thuyết trình mini 3 phút.

---

## Phiên 1: Ôn Tập Phát Âm (25 phút)

### 1.1 Word stress test
Đánh dấu trọng âm và đọc to:
1. authentication → au-then-ti-**CA**-tion ✓/✗
2. development → de-**VE**-lop-ment ✓/✗
3. asynchronous → a-**SYN**-chro-nous ✓/✗
4. repository → re-**PO**-si-to-ry ✓/✗
5. vulnerability → vul-ne-ra-**BI**-li-ty ✓/✗

### 1.2 Sentence stress drill
Đọc, nhấn đúng content words:
1. "The **CACHE** needs to be **CLEARED** **REGULARLY**."
2. "I'll **SET** up the **CI/CD** **PIPELINE** **TOMORROW**."
3. "The **BOTTLENECK** is in the **DATABASE** **QUERY**."

### 1.3 Connected speech review
Nối âm tự nhiên:
- "Check_it out." / "Set_it_up." / "Turn_it_off."
- "I'm gonna deploy." / "I hafta fix this."

---

## Phiên 2: Mini Presentation (30 phút)

### Bài tập lớn: Thuyết trình 3 phút
**Chủ đề:** Giới thiệu 1 dự án bạn đã/đang làm

**Cấu trúc:**
1. **Intro** (30 giây): Tên dự án, mục đích
2. **Tech stack** (1 phút): Công nghệ sử dụng
3. **Thử thách** (1 phút): Vấn đề gặp phải và cách giải quyết
4. **Kết** (30 giây): Kết quả và bài học

**Mẫu:**
> "Hi, I'd like to talk about a project I recently worked on. It's a [tên] application that [mục đích]. We built it using [tech stack]. The main challenge was [vấn đề]. We solved it by [giải pháp]. The result was [kết quả]. I learned a lot about [bài học]."

**Bước thực hiện:**
1. Viết script (10 phút)
2. Đọc to 3 lần (5 phút)
3. Nói KHÔNG nhìn (5 phút)
4. Ghi hình/ghi âm lần cuối (3 phút)
5. Xem/nghe lại → ghi chú cần cải thiện (7 phút)
