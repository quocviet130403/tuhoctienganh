## Phiên 1: Ôn Tập Phụ Âm (25 phút)

### 1.1 Test nhanh: Đọc to các từ sau
Đọc chậm, rõ ràng, chú ý âm được in đậm:

| Âm | Từ test |
|---|---|
| /θ/ | **th**ink, me**th**od, pa**th**, **th**read |
| /ð/ | **th**is, **th**at, **th**ey, toge**th**er |
| /ʃ/ | pu**sh**, ca**che**, func**tion**, applica**tion** |
| /tʃ/ | **ch**eck, bran**ch**, fea**ture**, fet**ch** |
| /dʒ/ | **J**avaScript, pa**ge**, ima**ge**, mana**ge** |
| /r/ | **r**un, **r**equest, er**r**or, a**rr**ay |
| /l/ | **l**ist, fi**l**e, too**l**, insta**ll** |
| /v/ | **v**ersion, en**v**ironment, **v**ariable |
| /f/ | **f**unction, **f**ile, **f**ix |
| /w/ | **w**eb, **w**ork, **w**indow |

Ghi âm toàn bộ → nghe lại → đánh giá.

### 1.2 Test đuôi -s (5 phút)
Đọc nhanh, phân biệt /s/ /z/ /ɪz/:
- test**s** /s/, code**s** /z/, page**s** /ɪz/
- stack**s** /s/, file**s** /z/, change**s** /ɪz/
- step**s** /s/, tool**s** /z/, fixe**s** /ɪz/

### 1.3 Test đuôi -ed (5 phút)
- push**ed** /t/, deploy**ed** /d/, start**ed** /ɪd/
- fix**ed** /t/, call**ed** /d/, updat**ed** /ɪd/
- crash**ed** /t/, merg**ed** /d/, creat**ed** /ɪd/
