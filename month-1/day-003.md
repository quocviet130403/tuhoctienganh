# Day 3: Nguyên Âm /ʌ/ vs /ɑː/ + Shadowing Cơ Bản

> 🎯 **Mục tiêu:** Phát âm đúng /ʌ/ và /ɑː/, bắt đầu kỹ thuật shadowing, học 15 từ vựng mới.

---

## Phiên 1: Phát Âm (25 phút)

### 1.1 Ôn tập nhanh (5 phút)
Đọc to nhanh: beat/bit, set/sat, bed/bad → cảm nhận sự khác biệt giữa các cặp âm.

### 1.2 Nguyên âm /ʌ/ — "uh" ngắn (10 phút)
- **Cách phát âm:** Miệng mở vừa, lưỡi ở vị trí trung tâm, âm ngắn gọn (giống "ơ" ngắn)
- **Ví dụ:**

| Từ | Phiên âm | Nghĩa |
|---|---|---|
| bug | /bʌɡ/ | lỗi (phần mềm) |
| run | /rʌn/ | chạy |
| up | /ʌp/ | lên |
| cut | /kʌt/ | cắt |
| function | /ˈfʌŋkʃən/ | hàm |
| button | /ˈbʌtən/ | nút |
| upload | /ˈʌploʊd/ | tải lên |
| number | /ˈnʌmbər/ | số |

### 1.3 Nguyên âm /ɑː/ — "a" dài (10 phút)
- **Cách phát âm:** Miệng mở rộng, lưỡi hạ thấp phía sau, âm dài (giống "a" trong tiếng Việt)
- **Ví dụ:**

| Từ | Phiên âm | Nghĩa |
|---|---|---|
| start | /stɑːrt/ | bắt đầu |
| hard | /hɑːrd/ | khó / cứng |
| part | /pɑːrt/ | phần |
| class | /klɑːs/ | lớp (BrE) |
| server | /ˈsɜːrvər/ | máy chủ |
| archive | /ˈɑːrkaɪv/ | lưu trữ |
| large | /lɑːrdʒ/ | lớn |
| target | /ˈtɑːrɡɪt/ | mục tiêu |

### 1.4 Minimal Pairs /ʌ/ vs /ɑː/

| /ʌ/ | /ɑː/ |
|---|---|
| cut /kʌt/ | cart /kɑːrt/ |
| hut /hʌt/ | heart /hɑːrt/ |
| luck /lʌk/ | lark /lɑːrk/ |
| duck /dʌk/ | dark /dɑːrk/ |
| cup /kʌp/ | carp /kɑːrp/ |

**Bài tập:** Đọc xen kẽ 10 lần.

## Phiên 3: Nghe & Shadowing (25 phút)

### 3.1 Shadowing là gì? (2 phút)
- **Shadowing** = nghe → lặp lại NGAY LẬP TỨC (chậm hơn 1-2 giây)
- Không cần hiểu 100%, tập trung bắt chước **ngữ điệu** và **nhịp điệu**

### 3.2 Thực hành Shadowing (15 phút)
- **Video:** YouTube — Fireship: "100+ Computer Science Concepts Explained" (chọn 2 phút đầu)
- **Cách làm:**
  1. Xem 1 lần có subtitle → hiểu nội dung
  2. Tua lại → shadowing từng câu (pause nếu cần)
  3. Lặp lại 3 lần không pause

### 3.3 Dictation (8 phút)
- Chọn 1 phút từ video trên
- Nghe → viết lại → so sánh với subtitle
- Ghi lại các từ nghe sai vào sổ tay

## Phiên 4: Tập Nói (10 phút)

### Mô tả code bằng tiếng Anh
Nhìn đoạn code dưới và tự giải thích bằng tiếng Anh:

```javascript
let name = "John";
let age = 25;
console.log(name);
```

**Ví dụ cách nói:**
- "First, I declare a variable called name and set it to the string John."
- "Then I declare another variable called age and set it to twenty-five."
- "Finally, I log the name to the console."

**Bài tập:** Ghi âm phần giải thích của bạn → nghe lại.
