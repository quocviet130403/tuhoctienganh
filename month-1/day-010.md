# Day 10: Phụ Âm /r/ vs /l/ — Lỗi Kinh Điển Người Việt

> 🎯 **Mục tiêu:** Phân biệt rõ ràng /r/ và /l/, sửa lỗi phổ biến nhất của người Việt khi nói tiếng Anh.

---

## Phiên 1: Phát Âm (30 phút)

### 1.1 Âm /l/ — "L" (10 phút)
- **Cách phát âm:** Đầu lưỡi ĐẬM CHẮC lên vòm miệng (phía sau răng cửa trên)
- **Light L** (đầu từ): lưỡi chạm vòm → bật ra
- **Dark L** (cuối từ): lưỡi chạm vòm → giữ → âm trầm hơn

| Từ | Phiên âm | Nghĩa | Vị trí L |
|---|---|---|---|
| list | /lɪst/ | danh sách | Light L |
| link | /lɪŋk/ | liên kết | Light L |
| load | /loʊd/ | tải | Light L |
| log | /lɒɡ/ | nhật ký | Light L |
| Linux | /ˈlɪnəks/ | Linux | Light L |
| tool | /tuːl/ | công cụ | Dark L |
| file | /faɪl/ | tệp | Dark L |
| null | /nʌl/ | null | Dark L |
| install | /ɪnˈstɔːl/ | cài đặt | Dark L |
| module | /ˈmɒdjuːl/ | module | Dark L |

### 1.2 Âm /r/ — "R" (10 phút)
- **Cách phát âm:** Lưỡi CONG LÊN TRÊN nhưng KHÔNG CHẠM vòm miệng
- **Mẹo:** Hãy tưởng tượng bạn đang sắp phát âm /l/ nhưng dừng lại trước khi lưỡi chạm
- **⚠️ KHÁC hoàn toàn với "r" tiếng Việt** (có rung)

|---|---|---|
| run | /rʌn/ | chạy |
| read | /riːd/ | đọc |
| write | /raɪt/ | viết |
| root | /ruːt/ | gốc |
| render | /ˈrendər/ | hiển thị |
| request | /rɪˈkwest/ | yêu cầu |
| response | /rɪˈspɒns/ | phản hồi |
| return | /rɪˈtɜːrn/ | trả về |

| error | /ˈerər/ | lỗi |

### 1.3 Minimal Pairs /r/ vs /l/

| /r/ | /l/ |
|---|---|
| right /raɪt/ | light /laɪt/ |
| read /riːd/ | lead /liːd/ |
| rock /rɒk/ | lock /lɒk/ |
| race /reɪs/ | lace /leɪs/ |
| rest /rest/ | lest /lest/ |
| run /rʌn/ | lung /lʌŋ/ |
| rain /reɪn/ | lane /leɪn/ |
| red /red/ | led /led/ |
| rip /rɪp/ | lip /lɪp/ |
| row /roʊ/ | low /loʊ/ |

**Bài tập chuyên sâu:** 
1. Đọc cột /r/ từ trên xuống → đọc cột /l/ từ trên xuống
2. Đọc ngang: right → light → right → light (mỗi cặp 5 lần)
3. Ghi âm → nghe lại → bạn có phân biệt được không?

## Phiên 4: Tập Nói (10 phút)

### Tongue twister (Câu luyện lưỡi)
Đọc to, chậm rõ ràng:

1. "**R**ed **l**orry, ye**ll**ow **l**orry" (nhanh dần)
2. "The **r**e**l**iab**l**e **r**e**l**ease was **r**eady for **l**aunch."
3. "A **r**ea**l** deve**l**oper **r**egu**l**ar**l**y **r**eviews the **l**og fi**l**es."

**Bài tập:** Mỗi câu đọc 5 lần → ghi âm lần cuối.
