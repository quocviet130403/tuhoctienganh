# Day 17: Connected Speech — Nối Âm & Nuốt Âm

> 🎯 **Mục tiêu:** Hiểu cách người bản ngữ nối và nuốt âm, giúp nghe hiểu tốt hơn.

---

## Phiên 1: Connected Speech (30 phút)

### 1.1 Linking — Nối âm (10 phút)
Khi phụ âm cuối gặp nguyên âm đầu → nối liền:

| Viết | Nói thành |
|---|---|
| check it | "che-kit" |
| turn off | "tur-noff" |
| set up | "se-tup" |
| an error | "a-nerror" |
| look at | "loo-kat" |
| run it | "ru-nit" |
| log in | "lo-gin" |
| shut it down | "shu-tit-down" |

**Bài tập:** Đọc to cả cụm, nối liền, KHÔNG ngắt giữa 2 từ.

### 1.2 Reduction — Rút gọn (10 phút)
Người bản ngữ rút gọn function words:

| Đầy đủ | Rút gọn | Ví dụ |
|---|---|---|
| want to | "wanna" | I wanna fix this bug. |
| going to | "gonna" | I'm gonna deploy. |
| have to | "hafta" | I hafta check the logs. |
| got to | "gotta" | I gotta push the code. |
| kind of | "kinda" | It's kinda slow. |
| a lot of | "alotta" | There are alotta errors. |
| don't know | "dunno" | I dunno what's wrong. |

**⚠️ Lưu ý:** Biết để NGHE HIỂU, nhưng trong phỏng vấn nên nói dạng đầy đủ!

### 1.3 Elision — Nuốt âm (10 phút)
Một số âm bị "nuốt" khi nói nhanh:

| Viết | Phát âm thực tế |
|---|---|
| next step | "neks-step" (t biến mất) |
| last time | "las-time" |
| must be | "mus-be" |
| test case | "tes-case" |
| first thing | "firs-thing" |

**Bài tập:** Nghe 1 video IT → đánh dấu chỗ nào bạn nghe thấy linking/reduction/elision.

## Phiên 3: Nghe (20 phút)
- Video tech 10 phút → nghe tốc độ 1x
- Ghi lại 5 chỗ có linking/reduction
- Dictation 1 phút

## Phiên 4: Tập Nói (10 phút)
Nói tự nhiên với linking:
> "I need to **set_up** the endpoint and **check_it** works. If there's **an_error**, I'll **look_at** the logs."

Ghi âm → so sánh với cách nói trong video.
