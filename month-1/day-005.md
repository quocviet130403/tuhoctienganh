# Day 5: Nguyên Âm /ʊ/ vs /uː/ + Ôn Tập Giữa Tuần

> 🎯 **Mục tiêu:** Phát âm đúng /ʊ/ và /uː/, ôn tập tổng hợp tất cả nguyên âm đã học, củng cố từ vựng.

---

## Phiên 1: Phát Âm (25 phút)

### 1.1 Nguyên âm /ʊ/ — "u" ngắn (10 phút)
- **Cách phát âm:** Môi tròn nhẹ, lưỡi kéo về phía sau, âm ngắn
- **Ví dụ:**

| Từ | Phiên âm | Nghĩa |
|---|---|---|
| put | /pʊt/ | đặt |
| push | /pʊʃ/ | đẩy (git push) |
| pull | /pʊl/ | kéo (git pull) |
| look | /lʊk/ | nhìn |
| book | /bʊk/ | sách |
| good | /ɡʊd/ | tốt |
| full | /fʊl/ | đầy |
| could | /kʊd/ | có thể |

### 1.2 Nguyên âm /uː/ — "oo" dài (10 phút)
- **Cách phát âm:** Môi tròn chúm nhỏ, lưỡi kéo về sau và lên cao, âm dài

| Từ | Phiên âm | Nghĩa |
|---|---|---|
| tool | /tuːl/ | công cụ |
| do | /duː/ | làm |
| group | /ɡruːp/ | nhóm |
| loop | /luːp/ | vòng lặp |
| root | /ruːt/ | gốc / root |
| move | /muːv/ | di chuyển |
| include | /ɪnˈkluːd/ | bao gồm |
| use | /juːz/ | sử dụng |

### 1.3 Minimal Pairs /ʊ/ vs /uː/

| /ʊ/ | /uː/ |
|---|---|
| pull /pʊl/ | pool /puːl/ |
| full /fʊl/ | fool /fuːl/ |
| look /lʊk/ | Luke /luːk/ |
| could /kʊd/ | cooed /kuːd/ |
| push /pʊʃ/ | poosh | (tưởng tượng)

**Bài tập:** Đọc xen kẽ 10 lần: pull → pool → pull → pool...

### 1.4 Ôn tập TỔNG HỢP tất cả nguyên âm (5 phút)

Đọc to mỗi từ 2 lần:
- /iː/ beat → /ɪ/ bit
- /e/ set → /æ/ sat
- /ʌ/ cut → /ɑː/ cart
- /ɒ/ not → /ɔː/ port
- /ʊ/ pull → /uː/ pool

## Phiên 4: Tập Nói — Mô tả Git workflow (15 phút)

### Tự nói (ghi âm):
Hãy giải thích một git workflow đơn giản:

**Mẫu:**
> "First, I clone the repository from GitHub. Then I create a new branch for my feature. I write code and make commits. When I'm done, I push my branch to the remote. Then I create a pull request. After code review, I merge the branch into main."

**Bài tập:**
1. Đọc mẫu trên 3 lần
2. Tự nói KHÔNG nhìn mẫu → ghi âm
3. Nghe lại → so sánh → làm lại nếu cần
