# Day 25: Câu Điều Kiện + Modal Verbs

> 🎯 **Mục tiêu:** Dùng If-clauses và modal verbs để thảo luận giải pháp kỹ thuật.

---

## Phiên 1: Câu Điều Kiện (25 phút)

### 1.1 If (Type 1) — Có thể xảy ra
**Cấu trúc:** If + Present Simple, will + V

| Câu ví dụ IT |
|---|
| **If** the test **passes**, I **will** deploy the code. |
| **If** we **add** caching, the app **will** be faster. |
| **If** the server **crashes**, we **will** lose data. |
| **If** you **don't** back up, you **will** regret it. |

### 1.2 If (Type 2) — Giả định / đề xuất
**Cấu trúc:** If + Past Simple, would + V

| Câu ví dụ IT |
|---|
| **If** I **had** more time, I **would** refactor the code. |
| **If** we **used** microservices, it **would** scale better. |
| **If** I **were** you, I **would** add error handling. |
| **If** we **switched** to TypeScript, we **would** have fewer bugs. |

---

## Phiên 2: Modal Verbs (25 phút)

### Các modal verbs quan trọng cho IT

| Modal | Ý nghĩa | Ví dụ IT |
|---|---|---|
| **can** | Có thể | I **can** write Python and JavaScript. |
| **could** | Có thể (lịch sự / giả định) | We **could** use Redis for caching. |
| **should** | Nên | You **should** write unit tests. |
| **would** | Sẽ (giả định) | I **would** recommend using TypeScript. |
| **might** | Có thể (chưa chắc) | This **might** cause a memory leak. |
| **must** | Phải | You **must** validate user input. |
| **need to** | Cần | We **need to** fix this bug before release. |

### Mẫu câu thảo luận giải pháp:
1. "I **would suggest** using a message queue."
2. "We **should** consider caching the results."
3. "This approach **could** improve performance by 50%."
4. "We **might** need to scale horizontally."
5. "If I **could** redesign this, I **would** use a different pattern."

## Phiên 3: Từ Vựng — Decision Making (15 phút)

| # | Từ | Nghĩa |

| 1 | approach | cách tiếp cận |
| 2 | alternative | thay thế |
| 3 | trade-off | đánh đổi |
| 4 | advantage | lợi thế |
| 5 | disadvantage | bất lợi |
| 6 | consider | xem xét |
| 7 | recommend | đề xuất |
| 8 | compare | so sánh |
| 9 | evaluate | đánh giá |
| 10 | implement | triển khai |
