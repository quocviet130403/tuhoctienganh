# Day 23: Thì Quá Khứ Đơn — Kể Kinh Nghiệm

> 🎯 **Mục tiêu:** Sử dụng Past Simple để kể về kinh nghiệm, dự án đã làm — kỹ năng cần thiết cho phỏng vấn.

---

## Phiên 1: Ngữ Pháp (30 phút)

### 1.1 Past Simple — Hành động đã hoàn thành
**Cấu trúc:** S + V-ed (regular) hoặc V2 (irregular) | S + didn't + V

### 1.2 Động từ IT thường gặp (Past Simple)

#### Regular verbs (-ed):
| Base | Past | Ví dụ |
|---|---|---|
| deploy | deployed | I **deployed** the app yesterday. |
| fix | fixed | She **fixed** the bug. |
| create | created | We **created** a new branch. |
| test | tested | I **tested** all features. |
| push | pushed | He **pushed** the code. |
| install | installed | I **installed** Node 18. |
| update | updated | We **updated** the database. |
| implement | implemented | I **implemented** the API. |

#### Irregular verbs (V2):
| Base | Past | Ví dụ |
|---|---|---|
| write | wrote | I **wrote** a script. |
| build | built | We **built** a dashboard. |
| run | ran | The test **ran** successfully. |
| find | found | I **found** the root cause. |
| make | made | She **made** a pull request. |
| send | sent | I **sent** the report. |
| break | broke | The update **broke** the API. |
| choose | chose | We **chose** React. |
| get | got | I **got** an error. |
| set | set | He **set** up the server. |

### 1.3 Câu phỏng vấn mẫu (Past Simple)
1. "In my **last** project, I **worked** on the backend API."
2. "I **built** a real-time chat application using WebSocket."
3. "We **migrated** from a monolith to microservices."
4. "I **found** a critical security vulnerability and **fixed** it."
5. "The team **delivered** the project on time."

### 1.4 Bài tập
Viết 10 câu Past Simple về kinh nghiệm của bạn:
- 5 câu regular verbs
- 5 câu irregular verbs
