## Phiên 1: Phát Âm (30 phút)

### 1.1 Cụm phụ âm đầu từ (10 phút)

| Cụm | Ví dụ |
|---|---|
| /str/ | string /strɪŋ/, stream /striːm/ |
| /skr/ | screen /skriːn/, script /skrɪpt/ |
| /spr/ | spread /spred/, sprint /sprɪnt/ |
| /pl/ | plan /plæn/, platform /ˈplætfɔːrm/ |
| /br/ | branch /bræntʃ/, browser /ˈbraʊzər/ |
| /tr/ | try /traɪ/, track /træk/ |
| /dr/ | drive /draɪv/, drop /drɒp/ |

**Mẹo:** Không thêm nguyên âm phụ! "string" = 1 âm tiết.

### 1.2 Cụm phụ âm cuối từ (10 phút)

| Cụm | Ví dụ |
