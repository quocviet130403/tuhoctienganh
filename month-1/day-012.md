# Day 12: Đuôi -ed + Phụ Âm /v/ /f/ /w/

> 🎯 **Mục tiêu:** Học 3 cách phát âm đuôi -ed, phân biệt /v/ /f/ /w/ — các âm người Việt hay nhầm.

---

## Phiên 1: Phát Âm (30 phút)

### 1.1 Đuôi -ed: Ba cách phát âm (15 phút)

| Phát âm | Khi nào | Ví dụ |
|---|---|---|
| **/t/** | Sau âm vô thanh: /p/ /k/ /f/ /s/ /ʃ/ /tʃ/ | stopped, checked, pushed |
| **/d/** | Sau âm hữu thanh và nguyên âm | called, used, played |
| **/ɪd/** | Sau /t/ hoặc /d/ | started, needed, updated |

#### Nhóm /t/:
- fixed /fɪkst/, pushed /pʊʃt/, checked /tʃekt/, stopped /stɒpt/, crashed /kræʃt/
- Luyện: "I **fixed** the bug." "She **pushed** the code." "We **checked** the logs."

#### Nhóm /d/:
- called /kɔːld/, solved /sɒlvd/, deployed /dɪˈplɔɪd/, changed /tʃeɪndʒd/, merged /mɜːrdʒd/
- Luyện: "I **called** the API." "He **deployed** the app." "They **merged** the branch."

#### Nhóm /ɪd/:
- started /ˈstɑːrtɪd/, updated /ʌpˈdeɪtɪd/, needed /ˈniːdɪd/, created /kriˈeɪtɪd/, loaded /ˈloʊdɪd/
- Luyện: "The server **started**." "I **updated** the file." "She **created** a new branch."

**Bài tập:** Phân loại các từ → đọc to → ghi âm.

### 1.2 Phụ âm /f/ vs /v/ (8 phút)
- **/f/** (vô thanh): răng trên chạm môi dưới → thổi hơi → KHÔNG rung
- **/v/** (hữu thanh): giống /f/ nhưng CÓ rung cổ họng
- **Lỗi người Việt:** Hay nói /v/ thành /w/ hoặc "dờ"

| /f/ | /v/ |
|---|---|
| file /faɪl/ | value /ˈvæljuː/ |
| function /ˈfʌŋkʃən/ | variable /ˈveriəbl/ |
| fix /fɪks/ | version /ˈvɜːrʒən/ |
| few /fjuː/ | view /vjuː/ |
| life /laɪf/ | live /lɪv/ |
| safe /seɪf/ | save /seɪv/ |

### 1.3 Phụ âm /w/ (7 phút)
- **Cách phát âm:** Môi chúm tròn rồi bật ra → KHÁC hoàn toàn /v/
- **⚠️ /w/ ≠ /v/:** "wine" ≠ "vine", "west" ≠ "vest"

| /w/ | /v/ |
|---|---|
| web /web/ | version /ˈvɜːrʒən/ |
| work /wɜːrk/ | verify /ˈverɪfaɪ/ |
| window /ˈwɪndoʊ/ | view /vjuː/ |
| wire /waɪər/ | virus /ˈvaɪrəs/ |
| wait /weɪt/ | validate /ˈvælɪdeɪt/ |

**Bài tập:** Đọc: web → version → web → version (5 lần). Cảm nhận: /w/ = môi tròn, /v/ = răng chạm môi.

## Phiên 4: Tập Nói (10 phút)

### Kể lại quy trình deploy
> "Yesterday, I **fixed** a bug and **committed** the code. Then I **pushed** it to the repository. The CI/CD **pipeline** **started** automatically. After the tests **passed**, the app was **deployed** to staging. We **checked** everything and then **moved** it to production."

Đọc 2 lần → nói không nhìn → ghi âm → kiểm tra tất cả đuôi -ed.
