# Day 22: Thì Hiện Tại Đơn & Tiếp Diễn — Mô Tả Công Việc

> 🎯 **Mục tiêu:** Sử dụng Present Simple và Present Continuous để mô tả công việc IT hàng ngày.

---

## Phiên 1: Ngữ Pháp (30 phút)

### 1.1 Present Simple — Thói quen, sự thật
**Dùng khi:** Nói về thói quen, quy trình cố định, sự thật chung.

**Cấu trúc:** S + V(s/es) | S + don't/doesn't + V

| Câu ví dụ IT | Giải thích |
|---|---|
| I **write** code every day. | Thói quen |
| The server **runs** on port 8080. | Sự thật |
| We **use** Git for version control. | Quy trình |
| The function **returns** a string. | Mô tả code |
| I **don't** use Windows for development. | Phủ định |
| **Does** the API **support** pagination? | Câu hỏi |

### 1.2 Present Continuous — Đang làm, tạm thời
**Dùng khi:** Đang làm ngay bây giờ, tình huống tạm thời.

**Cấu trúc:** S + am/is/are + V-ing

| Câu ví dụ IT | Giải thích |
|---|---|
| I**'m working** on the login feature. | Đang làm |
| We**'re migrating** to a new database. | Tạm thời |
| The team **is testing** the new release. | Đang diễn ra |
| I**'m not using** React for this project. | Phủ định |
| **Are** you **deploying** to production? | Câu hỏi |
| The server **is crashing** frequently. | Vấn đề hiện tại |

### 1.3 So sánh trong ngữ cảnh IT

| Present Simple | Present Continuous |
|---|---|
| I **work** as a developer. (nghề) | I**'m working** from home today. (hôm nay) |
| We **deploy** every Friday. (thói quen) | We**'re deploying** right now. (ngay bây giờ) |
| This API **uses** REST. (thiết kế) | I**'m using** Postman to test it. (lúc này) |

### 1.4 Bài tập viết
Viết 10 câu về công việc IT của bạn:
- 5 câu Present Simple (thói quen)
- 5 câu Present Continuous (đang làm)

Ghi âm → nghe lại.
