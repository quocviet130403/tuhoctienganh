# Day 6: Nguyên Âm Đôi /eɪ/ /aɪ/ /ɔɪ/ + Nghe Video IT

> 🎯 **Mục tiêu:** Phát âm đúng 3 nguyên âm đôi (diphthongs), nghe video IT dài hơn, mở rộng từ vựng.

---

## Phiên 1: Phát Âm (25 phút)

### 1.1 Nguyên âm đôi (Diphthongs) là gì? (3 phút)
- **Diphthong** = 2 nguyên âm nối liền trong 1 âm tiết
- Miệng bắt đầu ở vị trí âm 1 → trượt sang vị trí âm 2
- Tiếng Việt cũng có nguyên âm đôi: "ai", "ôi", "ao"

### 1.2 Nguyên âm đôi /eɪ/ — "ay" (7 phút)
- **Cách phát âm:** Bắt đầu từ /e/ → trượt lên /ɪ/

| Từ | Phiên âm | Nghĩa |
|---|---|---|
| make | /meɪk/ | tạo |
| name | /neɪm/ | tên |
| save | /seɪv/ | lưu |
| page | /peɪdʒ/ | trang |
| data | /ˈdeɪtə/ | dữ liệu |
| database | /ˈdeɪtəbeɪs/ | cơ sở dữ liệu |
| create | /kriˈeɪt/ | tạo mới |
| update | /ʌpˈdeɪt/ | cập nhật |

### 1.3 Nguyên âm đôi /aɪ/ — "eye" (7 phút)
- **Cách phát âm:** Bắt đầu từ /a/ → trượt lên /ɪ/

| Từ | Phiên âm | Nghĩa |
|---|---|---|
| file | /faɪl/ | tệp |
| type | /taɪp/ | loại / gõ |
| write | /raɪt/ | viết |
| line | /laɪn/ | dòng |
| find | /faɪnd/ | tìm |
| time | /taɪm/ | thời gian |
| compile | /kəmˈpaɪl/ | biên dịch |
| API | /ˌeɪ.piːˈaɪ/ | giao diện lập trình |

### 1.4 Nguyên âm đôi /ɔɪ/ — "oy" (5 phút)
- **Cách phát âm:** Bắt đầu từ /ɔː/ → trượt lên /ɪ/

| Từ | Phiên âm | Nghĩa |
|---|---|---|
| point | /pɔɪnt/ | điểm |
| join | /dʒɔɪn/ | tham gia / nối |
| deploy | /dɪˈplɔɪ/ | triển khai |
| choice | /tʃɔɪs/ | lựa chọn |
| avoid | /əˈvɔɪd/ | tránh |
| exploit | /ɪkˈsplɔɪt/ | khai thác (lỗ hổng) |

### 1.5 Đọc tổng hợp (3 phút)
Đọc nhanh cả 3 nhóm: make → file → point → save → write → join...

## Phiên 4: Tập Nói (10 phút)

### Giải thích cấu trúc dữ liệu
Chọn 1 cấu trúc dữ liệu và giải thích bằng tiếng Anh:

**Mẫu (Array):**
> "An array is a data structure that stores a collection of elements. Each element has an index, starting from zero. You can access any element by its index. Arrays are good for fast access but slow for inserting or removing elements in the middle."

**Bài tập:** Giải thích 1 cấu trúc dữ liệu khác (stack, queue, hoặc tree) → ghi âm.
