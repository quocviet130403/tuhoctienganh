## Phiên 1: Phát Âm — Nguyên âm đôi còn lại (20 phút)

Đọc to mỗi từ, tự kiểm tra xem bạn phát âm đúng chưa:
1. **feature** /ˈfiːtʃər/ → /iː/
2. **git** /ɡɪt/ → /ɪ/
3. **test** /test/ → /e/
4. **app** /æp/ → /æ/
5. **bug** /bʌɡ/ → /ʌ/
6. **start** /stɑːrt/ → /ɑː/
7. **log** /lɒɡ/ → /ɒ/
8. **port** /pɔːrt/ → /ɔː/
9. **push** /pʊʃ/ → /ʊ/
10. **tool** /tuːl/ → /uː/
11. **data** /ˈdeɪtə/ → /eɪ/
12. **file** /faɪl/ → /aɪ/
13. **deploy** /dɪˈplɔɪ/ → /ɔɪ/
14. **cloud** /klaʊd/ → /aʊ/
15. **code** /koʊd/ → /əʊ/
16. **engineer** /ˌendʒɪˈnɪər/ → /ɪə/

### 4.2 ELSA Speak review (10 phút)
- Vào ELSA → làm bài test nguyên âm
- So sánh kết quả với Day 1 → ghi lại tiến bộ

### 4.3 Ghi âm tổng hợp (10 phút)
Ghi âm bản thân nói đoạn sau (KHÔNG nhìn):
> "Hi, my name is [tên]. I am a software developer. I use tools like VS Code and Git every day. I write code, debug errors, and deploy applications. I want to improve my English to prepare for IT job interviews."

Nghe lại → đánh giá phát âm → lưu lại để so sánh sau 1 tháng.
